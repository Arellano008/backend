Backend y Entorno Virtual

Un entorno virtual en Python es una herramienta que permite aislar las dependencias de un proyecto del entorno global del sistema. Esto es especialmente útil cuando trabajas en múltiples proyectos con diferentes requerimientos de paquetes y versiones.

¿Por qué usar un entorno virtual?

  • Evita conflictos de versiones entre proyectos.
  • Facilita la portabilidad del proyecto.
  • Mantiene el entorno global limpio.

Pasos para Configurar un Entorno Virtual y Flask

1. Crear un entorno virtual

Ejecuta el siguiente comando para crear un entorno virtual:

python -m venv nombre_del_entorno

Si no aparece nada en el terminal, significa que se ha creado exitosamente.

2. Activar el entorno virtual

Para activarlo, utiliza el siguiente comando:

scripts/activate

Una vez activado, verás el nombre del entorno al inicio del prompt.

3. Instalar Flask

Una vez activado el entorno, instala Flask con:

pip install Flask
4. Ejecutar la aplicación

Ejecuta tu aplicación con el siguiente comando:

python app.py

Asegúrate de verificar que todos los enlaces funcionen correctamente en tu HTML.
